Teste JSF
  1. Criar a tela conforme exemplo *
  2. Ao carregar a tela, buscar os dados no banco de dados ordenados pelo nome
  3. Validar para não permitir cadastrar uma pessoa com o mesmo nome, exibir mensagem quando encontrar um nome já existente
  4. Todos os campos são de preenchimento obrigatório
  5. Salvar no banco de dados M - para masculino e F - para feminino
  6. Exibir na tabela a idade da pessoa de acordo com a sua data de nascimento
  7. Adicionar um campo de endereço na Pessoa, mapeando com JPA
  8. Consultar a API https://brasilapi.com.br/ bara buscar os dados de endereço pelo CEP
  9. Você deverá configurar o servidor Glassfish no Eclipse http://download.java.net/glassfish/4.1.2/release/glassfish-4.1.2.zip
  10. Você deverá criar uma branch com o seu nome
  11. Você deverá criar um commit para cada item desenvolvido
  12. O projeto esta configurado para utilizir o banco H2 em memória, ou seja, ao subir a aplicação será criado o banco de dados e a tabela do teste
  13. Abrir um Pull Request com as alterações realizadas
